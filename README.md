# Supreme Commerce

Marketing website for Supreme Commerce, a full-service e-commerce team that builds and runs Walmart, Facebook Marketplace, TikTok Shop, and Shopify stores — plus the website, SEO, and social media around them — for growing brands under fixed pricing.

## Pages

- `index.html` — home page
- `about.html` — about the team/founder
- `services.html` — services offered
- `portfolio.html` — portfolio/work samples
- `contact.html` — contact page

## Structure

Static site built with plain HTML, CSS, and JavaScript.

```
assets/
  style.css, responsive.css   # styles
  script.js, responsive.js    # behavior
  chat-widget.js               # chat widget
  images/, videos/              # media assets
```

## Running locally

Open any `.html` file directly in a browser, or serve the folder with a local static server, e.g.:

```
npx serve .
```
